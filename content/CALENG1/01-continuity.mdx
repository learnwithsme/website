| Continuous | if
| --- | --- |   
| on open interval $(a,b)$ | if continuous at each point of the interval
| everywhere | if continuous on $(-\infty, +\infty)$
| at a point $a$ | if $f(a)$ is defined,<br/>$\lim_{x\to a} f(x)$ exists, and<br/>$\lim_{x\to a}f(x)=f(a)$
| | 
| all polynomial functions | continuous everywhere
| rational expressions | continuous everywhere except those that make the denominator = 0
| radicals $\sqrt[n]{x}$ ($n$ is a positive integer) | if $n$ odd, continuous everywhere<br/>if $n$ even, continuous at all positive numbers
| absolute value $\lvert x\rvert$ | continuous everywhere

## How to classify

1. Identify value of $x$
   * Equating denominator to $0$
   * Identifying break in unit-step function
   * Remove values not included in the array of domains
2. Evaluate $\lim$ of each $x$ value
   * Check infinite limit, left-hand & right-hand limit
   * Evaluate indeterminate form
3. Classify
